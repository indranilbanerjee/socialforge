# Reactive Post

Create an unplanned post in response to a trending topic, news event, or opportunity.

## Process
1. Accept topic and context from user
2. Determine target platforms (default: all active)
3. Assign as HYGIENE tier (fast approval) unless user specifies otherwise
4. Generate creative using STYLE_REFERENCED or PURE_CREATIVE mode
5. Adapt copy for platforms
6. Fast-track through compliance check
7. Add to calendar-data.json as an extra post
8. Show for immediate approval
